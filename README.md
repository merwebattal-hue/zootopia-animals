# Zootopia Animals

This project generates an HTML page using Python, JSON data, and an HTML template (Jinja2).

## How it works
- `animals_data.json` → data source
B- `animals_template.html` → HTML template
- `animals_web_generator.py` → generator script
- `animals.html` → generated output

## Run
```bash
python animals_web_generator.py
